一、目录结构

-src -main -java -package #主函数，启动类，运行它如果运行了 Tomcat、Jetty、Undertow 等容器 -SpringbootApplication -resouces #文件夹-存放静态资源 js/css/images 等 -static #文件夹-存放 html 模板文件 -templates #文件-主要配置文件，SpringBoot启动时候会自动加载application.yml/application.properties -application.yml #测试文件存放目录 -test #pom.xml 文件是maven构建的寄出，里面包含了我们所依赖JAR和Plugin的信息 -pom.xml
